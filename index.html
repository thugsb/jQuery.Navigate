jQuery.Navigate

jQuery.Navigate simplifies single-page navigation, allowing for dual-layered content to be present elegantly and quickly. It allows entire (small) sites to be constructed as a single page, or larger sites to display masses of content in a user-friendly way.

1. Set up Dependencies

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.5.1/jquery.min.js" type="text/javascript"></script>
<script src="jquery.cycle.all.min.js" type="text/javascript"></script>
<script src="jquery.navigate.js" type="text/javascript"></script>

2. Call the Method

<script type="text/javascript">
jQuery(document).ready(function($) {
  $('#pageContainer').navigate();
});
</script>

3. Set up your Page

<div id="pageContainer">
  <div id="home">
    <h1>Home</h1>
  </div>
  <div id="gallery">
    <h2>Gallery</h2>
    <div class="subpages">
      <img id="picture1" src="images/tree1.jpg" alt="Tree1">
      <img id="picture2" src="images/tree2.jpg" alt="Tree2">
      <img id="picture3" src="images/tree3.jpg" alt="Tree3">
    </div>
  </div>
</div>

4. Include your Nav

<ul id="nav"C>
  <li><a href="#home">Home</a></li>
  <li><a href="#gallery">Gallery</a>
    <ul>
      <li><a href="#gallery/picture1">Picture 1</a></li>
      <li><a href="#gallery/picture2">Picture 2</a></li>
      <li><a href="#gallery/picture3">Picture 3</a></li>
    </ul>
  </li>
</ul>

Options and Defaults

$(element).navigate({
  'instantLoad'        : 0 // Set to 1 for the main page to appear initially without cycling
  'mainCycle'          : {fx:'blindX',timeout:0}, // jQuery.cycle JSON settings. Remember timeout:0
  'subCycle'           : {fx:'blindY',timeout:0}, // jQuery.cycle JSON settings. Remember timeout:0
  'subpageContainer'   : '.subpages', // Selector for container of subpages
});

Notes

The structure of the HTML must be as the above:

  • IDs of elements must be lowercase. I hope to remove this limitation with future releases.
  • The first character of the IDs must be a letter (not a number).
  • The first child elements within #pageContainer (or whatever element is called for navigating) must contain the IDs for the first layer of navigation.
  • The first child elements within the .subpages (or whatever you set as the subpageContainer) will be the IDs for the second layer of navigation (not yet tested with the jQuery.cycle slideExpr option).
  • Links should have the format href="#pageID/subpageID" or just href="#pageID". You cannot call directly to href="#subpageID" as this will break the page.

This plugin is based on the robust and flexible jQuery.Cycle, and the solid jQuery.hashchange (included). Thanks to Malsup and Ben Alman for maintaining these excellent plugins.

Compatible with jQuery 1.2.6+, jQuery.Navigate is tested on Firefox, Webkit and ie6-9 browsers.
